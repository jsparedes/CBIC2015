1. Introdução

1.1 Foco

  • Bases de dados de Clasificação
  • Os Sistemas de Inferência Fuzzy (SIF) surgem como uma alternativa que une:
    • capacidade de interpretação a nível linguístico
    • acurácia.

Nos Sistemas de Inferência Fuzzy

Na maior parte de trabalhos na literatura, eles misturam-se com os Algoritmos Evolucionários

1.2 Modelo AutoFIS

Procura-se:

  • Geração automática de SIF para Classificação
  • Sem uso de Algoritmos Evolucionários

Vantagens:

  • Mais deterministico
  • Procura-se a redução do tempo computacional

Disvantagens:

  • Modelo com muitos parametros.
  • Se a base de dados é grande, a base pode ser muito custosa.

4. Conclusões

Conclusões

  • Foi gerada uma nova metodologia para a geração automática de Sistemas de Inferência Fuzzy.
  • Nos experimentos obteve o segundo melhor resultado em termos de acurácia
  • Nos experimentos obteve o melhor resultado em termos de compactação da base de regras.
